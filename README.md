# Transcendence

## how to setup dev container:
```
git clone
```

```
cd <path to repo>
```

```
docker compose up
```
```
docker exec -it app bash
```
```
npm run build
```
```
npm run start
```
#### open another terminal
```
docker exec -it app bash
```

curl localhost:8080/ping
